[id="faq-revert-kojiversion"]
.How do I revert to an old version of my template?
[%collapsible]
====
To revert to a previous commit,

include::{includespath}/steps-codeeditor.adoc[tag=openproject-template2edit]

include::{includespath}/steps-codeeditor.adoc[tag=expand-terminal]

. Run `git log` and copy the hash code of the commit you want to restore.

. In the terminal panel, enter `git reset --hard <commit-hash>`.
+
WARNING: You will lose all the changes you made after the commit you restore.

TIP: Publish more frequently for a more detailed commit history.
====


[id="faq-reopen-frontend-terminal"]
.How do I reopen the frontend terminal?
[%collapsible]
====
To re-open the frontend terminal:

. Click on the [.tabname]`New` tab.

. Run:
+
....
cd frontend
npm start
....
====
